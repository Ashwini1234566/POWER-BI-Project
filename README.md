# POWER-BI-Project

Village-Level Crop & Flower Oversupply Prediction System

Project Overview

This project presents an interactive Crop & Flower Oversupply Prediction Dashboard built using Power BI. The dashboard analyzes crop and flower production, market demand, prices, and profit trends across different villages and seasons.

The goal of this project is to help farmers identify oversupply risks and choose profitable crops or flowers to plant, enabling better data-driven agricultural decisions.

⚙️ Tools & Technologies Used
Power BI Desktop – Dashboard development
Excel / CSV Dataset – Data source
Power Query – Data cleaning and transformation
DAX – KPI calculations and measures

🛠️ Steps Followed
Collected crop and flower production data including:
Market demand
Market supply
Market price
Production cost
Seasonal data
Village information
Imported dataset into Power BI
Cleaned and transformed the dataset using Power Query
Created calculated fields and measures such as:
Profit per kg
Production totals
Demand and supply comparison
Built interactive visualizations including:
Bar charts
Pie charts
Line charts
KPI cards
Designed an interactive dashboard layout to display insights clearly.

📈 Key Insights
Some flowers and crops show high supply but lower demand, leading to oversupply risk.
Certain flower types provide higher profit margins.
Production levels vary significantly across villages and seasons.
Farmers can reduce losses by selecting crops based on market demand trends.
